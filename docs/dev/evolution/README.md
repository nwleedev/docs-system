# Evolution Guidance

This folder owns future proposals, migration candidates, alternatives, technical debt, and recommended future patterns. Documents here are not current rules.

## Include

- `Recommended Future Pattern`, `Deferred`, and `Open Question` guidance that is not ready for current-rule promotion.
- Migration conditions, risks, rollback criteria, and promotion criteria.
- Development guidance targets whose owner folder is `evolution`.

## Exclude

- Rules that must apply to new code now. Use `architecture/`, `engineering/`, or `domain/`.
- Accepted long-lived decisions. Use `decisions/`.
- AI reading order or uncertainty behavior. Use `ai/`.

## Current Index

No detailed evolution proposal is present yet. Add one only when repository evidence supports a future direction but current-rule promotion is not justified.

## Promotion Rule

Future guidance becomes a current rule only after target repository evidence, verification, and any required decision record support the promotion.
